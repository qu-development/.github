# Cyvy Online

We build production software for companies that need operational systems, automation, AI-assisted workflows, and pragmatic infrastructure that can be deployed, observed, and maintained.

Cyvy Online is a Brazil-based software company. This GitHub organization is our public engineering profile for product and client delivery work.

## What We Build

### Product Showcase

- **Real-estate AI operations** - conversational lead handling, broker handoff workflows, internal monitoring, and mobile/desktop tools for real estate teams.
- **Sports and field-service mobile products** - store-ready mobile applications with subscriptions, payment gateways, and AI-assisted user experiences.
- **Compliance and operations software** - dense back-office systems for carriers, POS/customer service, security operations, reporting, and workflow automation.
- **Business automation** - WhatsApp automation, AI agents, dashboards, notifications, document/report generation, and production integrations.

### Production SaaS and AI Products

- **Hauzertech / Agente Hai** - a production real-estate AI-agent SaaS with WhatsApp-first lead handling, tenant-isolated operations, a Django backend, React web console, WinUI desktop operations app, .NET MAUI mobile app, Astro marketing surfaces, Firebase push notifications, GCP-backed operations, VPS deployment, backups, and monitoring.
- **TeeShot AI** - a Flutter mobile product approved on [Google Play](https://play.google.com/store/apps/details?id=com.qudevelopment.teeshot) and the [Apple App Store](https://apps.apple.com/us/app/teeshot-ai/id6759768465), with AI caddie features, GPS/course workflows, subscriptions, payment gateways, and release operations.
- **FleetKeeper** - carrier and DOT compliance software for operational reporting, compliance intelligence, and carrier-data workflows.
- **Q.Suite** - .NET customer-service and POS foundations for local operations, assisted workflows, and back-office tooling.

### Automation and Platform Work

- Multi-tenant Django and Django REST Framework backends.
- Celery, Redis, RabbitMQ, Kafka, workers, schedulers, and async processing.
- WhatsApp and messaging automation.
- AI-assisted business workflows and internal operator tools.
- Web, mobile, and desktop clients across TypeScript, React, Angular, C#, WinUI, .NET MAUI, and Dart.
- CI/CD, Docker, Nginx, Linux VPS operations, GCP, Firebase, backups, and release verification.

## Engineering Style

We prefer systems that are useful in production over demos that only look finished. Our work usually includes the boring but necessary parts: tenant safety, logs, deployment scripts, smoke checks, backups, permissions, operational documentation, and real follow-through after release.

## Main Technologies

Python, Django, Django REST Framework, Celery, AWS Lambda Powertools, PostgreSQL, Redis, RabbitMQ, Kafka, TypeScript, Angular, Blazor, React, C#, .NET, WinUI, .NET MAUI, Flutter, Dart, Docker, Nginx, GCP, Firebase, AWS, Linux, and GitHub Actions.
